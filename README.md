# MyASCII Pet üêæ

A console-based ASCII pet simulation written in C, allowing users to feed, play with, bathe, and care for a virtual pet. Includes a fun **Rock, Paper, Scissors, Lizard, Spock** minigame to play with your pet and maintain its happiness.  

---

## Features
- Interactive menu-driven console interface with input validation.
- Multiple pet types: Dog or Cat with ASCII portraits for actions and moods.
- Dynamic pet stats: Hunger, Happiness, Hygiene, Energy.
- Persistent data: Save and load pet state using a binary file.
- Minigame integration for enhanced user engagement.
- Clear and informative feedback for invalid inputs and game actions.

---

## Technologies
- **Languages:** C
- **Tools:** GCC, Makefile, Linux terminal

---

## Getting Started

### Prerequisites
- GCC compiler installed (Linux or Windows with MinGW)
- Terminal or command-line interface

### Build and Run
Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/MyASCII-Pet.git
cd MyASCII-Pet
```
Run the program:
```bash
make
./MyASCII-Pet
```
Clean up compiled files:
```bash
make clean
```
